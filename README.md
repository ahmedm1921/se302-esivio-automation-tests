# SE302 – Esivio Automated Testing Project

This repository contains an automated testing project for the Esivio website.

## Tools
- Playwright
- JavaScript
- Page Object Model (POM)

## Project Structure
- pages/ – Page Object classes
- tests/ – Functional and smoke tests

## Description
The project includes 15 automated test cases (10 functional and 5 smoke) implemented using Playwright and structured using the Page Object Model.
